Game

This game is a variation of the classic game Bulls and Cows.

Rules

The goal of the game is to guess the code. The code consists of digits from 0 to 9, the digits are not repeated. The player enters his guess and receives an answer in the format: the number of digits in the correct position and the number of correct digits in the wrong position.

Example

Code: 4321 Attempt: 0423 Result: 0423 1 2

Indicator

The indicator under the input field shows the result of the last attempt. Note that the colors of the indicator are not positionally related to the fields above it, but only display the number of correctly guessed positions and numbers.

Control

In addition to mouse control, keyboard control is available: Numeric keys - digit input Arrow keys - navigation through focused input fields Backspace - clear the field or shift focus to the previous one if the field is cleared Input - check the attempt
